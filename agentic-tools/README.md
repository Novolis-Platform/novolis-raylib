# Agentic tools

Registry and scripts for **AI agents** (and humans) working in this repo. Use these instead of improvising edits to generated files.

**Start here:** [registry.json](registry.json) (machine-readable) · [workflows/codegen.md](workflows/codegen.md) (step-by-step)

## Non-negotiable rules

1. **Never hand-edit** `*.g.cs` under `src/Novolis.Raylib.Bindings/Interop/` or `src/Novolis.Raylib.Runtime/` (files contain `<auto-generated />`).
2. **Manifests are source of truth** — `pipeline/raylib6/*.manifest.json`.
3. **After manifest or hook changes**, run `codegen.generate` then `agent.verify` before finishing the task.
4. **`src/`** = publishable NuGet packages only. **`codegen/`** = build tooling (never published).

## Quick commands

| Tool id | Command |
|---------|---------|
| `codegen.generate` | `dotnet run --project codegen/Novolis.Raylib.CodeGen -- generate` |
| `codegen.drift-check` | `pwsh ./scripts/raylib-codegen-check.ps1` |
| `agent.verify` | `pwsh ./scripts/agent-verify.ps1` |

## Cursor integration

- Hooks: `.cursor/hooks.json` (blocks direct edits to generated `*.g.cs`)
- Root guide: [AGENTS.md](../AGENTS.md)

## Adding a tool

Add an entry to `registry.json` and a one-line note in this README. Prefer wrapping existing `scripts/*.ps1` rather than duplicating logic.
